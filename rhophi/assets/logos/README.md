## Notes
- Recommended size: 200×200px or larger (square)
- Transparent background PNGs work best on dark backgrounds
- If a logo file is missing, the site falls back to displaying the company's ticker symbol
- To update a logo, just replace the file with the same filename — no code changes needed

## Updating Company Data
All company information is in `js/companies.js`. You can edit:
- Company name, tagline, description
- Valuation (performance index)
- Shares & float
- Profit/Loss history (6 periods)
- Market growth projections (5 years)
- Status: "growth" | "stable" | "watchlist"
- HQ location and founding year
